git config --global user.name blabla - введення імені користувача git config --global user.email blabal@bla - введеня e-mail адреси користувача git init - гіт "вмикається" для даного репозиторія git status - показує, які файли було змінено git add - додає файли в "commit" git commit - виконує "commit" змін git log - перераховує комміти, зроблені в репозиторії, в зворотньому хронологочному порядку (від останньої зміни) git checkout - дозволяє переміщатися між гілкаи. створеними командою "gt branch" git branch - команда для керування гілками в репозиторії git push - відправка проекту в з локального у віддалений репозиторій git pull - з віддаленого репозиторія завантажує інформацію на локальний git merge - об'єднує декілька послідовних коммітів в одну історію git clone - клонування локального чи віддаленого репозиторію. Клонування пустого репозиторію .gitignore - команда, що захищає від додавання непотірбних файлів в репозиторій
